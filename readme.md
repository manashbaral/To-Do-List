# To-Do-List
 
 ## Description
 A simple and beginner-friendly command-line To-Do List application written in Python.

 It allows user to add tasks, view tasks, update task status and delete tasks.

 All the tasks are stored in a text file(todo_list.txt).

---

 ## Features
 - Add new tasks.
 - View the to-do list in a formatted table.
 - Delete tasks by tasks name.
 - update task status(e.g., "incomplete" -> "complete").
 - Automatically stores and updates tasks in a text file.
 - Clean, simple code for beginners learning file handling.
